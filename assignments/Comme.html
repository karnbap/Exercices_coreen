Comment dire "COMME" en Coréen ? 🇰🇷

Le guide pour ne plus jamais se tromper !

La Règle d'Or 🔑

Pour savoir quel mot utiliser, pose-toi cette question : est-ce que "comme" est lié à un verbe (une action) ou à un nom (un objet) ?

🙋‍♀️

같아요

On l'utilise pour dire "Être comme...", quand on fait une simple comparaison.

Exemple :

"Elle est comme un ange."

→ 천사 같아요.

🏃‍♂️

처럼 / 같이

On les utilise pour décrire la manière de faire une action (avec un verbe). Ils sont souvent interchangeables.

Exemple :

"Je mange comme une vache."

→ 소처럼 먹어요 / 소같이 먹어요.

⚠️ Attention avec 같이 !

같이 peut aussi vouloir dire "ensemble". 처럼 ne peut pas !

"Je suis allé avec mon ami."

→ 친구랑 같이 갔어요. (Correct)

→ 친구처럼 갔어요. (Faux)

✍️

같은

On l'utilise pour décrire un nom. C'est comme dire "qui est comme...".

Exemple :

"C'est une amie comme ma famille."

→ 가족 같은 친구예요.

(Une amie QUI EST COMME ma famille)

🤔 Un cas un peu plus difficile : La danse !

별 같은 춤

Ici, 같은 décrit le nom "danse" (춤).

→ "Une danse qui est comme une étoile." (On parle du type de danse).

별 같이 춤을 춰요

Ici, 같이 décrit le verbe "danser" (춤을 춰요).

→ "Elle danse à la manière d'une étoile." (On parle de la façon de danser).

En Résumé 📝
