“Comme” — 무엇을 ‘처럼/같이/같다’로 말하나요?

FR. Comme peut: 1) qualifier un nom (comme + nom) → comparaison d’un nom ; 2) s’employer après être (être comme …) → identité/assimilation ; 3) modifier un verbe (verbe + comme …) → manière (« courir comme le vent »).

  • nom + comme → 명사 + 처럼/같이
  • être + comme → ~와/과 같다
  • verbe/adj + comme → 동사 + 처럼/같이
