“Comme” — Je décris quoi ? / 무엇을 ‘처럼/같이/같다’로 말하나요?

Comme… décrit quoi ? → un verbe (une action) ou un nom ou le verbe “être” ?

  • nom + comme → 명사 + 처럼/같이
  • être + comme → ~와/과 같다
  • verbe/adj + comme → 동사 + 처럼/같이
