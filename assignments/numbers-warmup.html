📘 Apprendre les nombres coréens ! / 한국어 숫자 배우기!

(Ft. 한자(Hanja = écriture) vs chinois (la langue) ?!)

1️⃣ Petite explication / 간단 설명

👉 Beaucoup de Français pensent que « 한자 = chinois ». Mais non ! / 많은 프랑스인이 ‘한자 = 중국어(언어)’라고 생각하지만 아닙니다!

  • 한자 (Hanja) = système d’écriture / 글자(문자) 체계
  • Chinois = langue / 중국어(언어)

Astuce (analogie) / 비유

En Europe, on écrit avec l'alphabet latin (pas « alphabet italien »).
→ Italien = une langue / lettres = alphabet latin.
→ chinois = une langue / lettres = caractères han (Hanja).
🇰🇷 Le coréen aussi : langue = coréen / nombres = 2 jeux (순수 한국어 / natifs, 한자어 / Hanja-coréens).
Hanja ≠ chinois (la langue) → « caractères han » = écriture, pas la langue chinoise.

En coréen, on utilise deux systèmes de nombres / 한국어에는 숫자 체계가 두 가지 있어요:

순수 한국어 / Natifs

하나, 둘, 셋…

Pour / 용도: compter des choses (âge, personnes, objets…) / 눈앞에서 세는 것(나이, 사람 수, 물건 개수…)

한자어 / Hanja-coréens

일, 이, 삼…

Pour / 용도: noms/unités déjà écrits (dates, argent, minutes…) / 이미 적혀 있는 이름·단위(날짜, 돈, 분 등)

⚡ Exception & Tendance / 예외와 경향

  • Heures(시) → 순수 한국어 / natifs · Minutes(분) → 한자어 / Hanja-coréens
  • « 1 h = 60 min » 체계 정착 → 오늘날 시 = natifs / 분 = Hanja

✅ Résumé facile / 요약

Système / 체계 Usage / 용도 Exemples / 예 Explication (FR)
순수 한국어 / Natifs Compter, classificateurs, durées libres 한 달, 두 달 / 한 개, 두 명, 세 잔, 한 권 / 한 시, 두 시 👉 한 달 = un mois (durée)
👉 한 개 = une chose (objet)
👉 두 명 = deux personnes
👉 세 잔 = trois verres/tasses
👉 한 권 = un livre
👉 한 시 = 1 heure (il est 1h)
한자어 / Hanja-coréens Valeurs « écrites » 1월, 2월 / 10분, 30초 / 2025년 9월 7일, 1,000원 / 3층, 302호 👉 1월 = janvier
👉 10분 = 10 minutes
👉 30초 = 30 secondes
👉 2025년 9월 7일 = le 7 septembre 2025
👉 1,000원 = 1000 wons (monnaie)
👉 3층, 302호 = 3ᵉ étage, appartement 302
