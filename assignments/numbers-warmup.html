📘 Apprendre les nombres coréens ! / 한국어 숫자 배우기!

(Ft. 한자(Hanja = écriture) vs chinois (la langue) ?!)

1️⃣ Petite explication / 간단 설명

👉 Beaucoup de Français pensent que « 한자 = chinois ». Mais non ! / 많은 프랑스인이 ‘한자 = 중국어(언어)’라고 생각하지만 아닙니다!

  • 한자 (Hanja) = système d’écriture / 글자(문자) 체계
  • Chinois = langue / 중국어(언어)

Astuce (analogie) / 비유

En Europe, on écrit avec l'alphabet latin…
→ chinois = langue / lettres = caractères han (Hanja).
🇰🇷 Le coréen : 2 jeux de nombres (순수 한국어, 한자어).

Deux systèmes / 두 가지 숫자

순수 한국어 / Natifs

하나, 둘, 셋…

Pour: âge, personnes, objets… / 눈앞에서 세는 값

한자어 / Hanja-coréens

일, 이, 삼…

Pour: unités écrites (dates, argent, minutes…)

⚡ Exception & Tendance / 예외와 경향

  • Heures(시) → natifs · Minutes(분) → Hanja-coréens
  • « 1 h = 60 min » 체계 이후 시=natifs / 분=Hanja

✅ Résumé / 요약

Système / 체계 Usage / 용도 Exemples / 예
순수 한국어 / Natifs Valeurs libres, classificateurs, durées 한 달, 두 달 / 한 개, 두 명, 세 잔, 한 권 / 한 시, 두 시
한자어 / Hanja-coréens Libellés écrits (calendrier, monnaie, montre…) 1월, 2월 / 10분, 30초 / 3층, 302호 / 1,000원
