“Comme” — ‘처럼/같이/같다' 무엇을 꾸미나요? Qu'est ce que l'on qualifier avec ‘처럼/같이/같다' ?

FR. Comme peut avoir plusieurs usages :

  • qualifier un nom → comme + nom = comparaison d’un nom
  • s’employer après être → être comme … = identité / assimilation
  • modifier un verbe → verbe + comme … = manière (ex: courir comme le vent)
nom + comme 명사 + 같은
être + comme ~와/과 같다
verbe/adj + comme 동사 + 처럼/같이
